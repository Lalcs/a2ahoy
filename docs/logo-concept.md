# Logo Concept

## Name

A2Ahoy = A2A (Agent-to-Agent) + Ahoy (a maritime greeting)

A CLI tool for checking connectivity between agents — a metaphor for calling out, "Ahoy! Can you hear me?"

## Symbol — Anchor + Signal

**Anchor (⚓):** A nautical motif representing "Ahoy." It symbolizes a stable foundation and the reliability of established connections.

**Signal Ring (green):** A status indicator for successful connectivity. It evokes the image of a ping response radiating outward. Positioned to the upper-right of the anchor, adding motion and dynamism to the otherwise static symbol.

## Typography

- `A2A` = Blue (#1A73E8) — emphasizes the protocol name
- `hoy` = Dark (#2D2D2D) — conveys friendliness and approachability of the greeting

Inter Semi Bold, -1.5px tracking for a modern look.

## Color Palette

- **Blue #1A73E8** — A trusted blue inspired by the A2A protocol (Google-originated)
- **Green #34A853** — A signal for successful connection and health check OK
- **Yellow #FBBC04** — An accent color for caution and warnings

These three colors also map to CLI output status displays.

## Decorative Dots

Three dots in green, blue, and yellow represent a terminal-style status lamp.

They suggest three levels of connectivity checks: success / info / warning.
